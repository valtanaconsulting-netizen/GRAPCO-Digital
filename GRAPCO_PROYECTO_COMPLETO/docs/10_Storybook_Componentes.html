📊 CPI (Cost Performance Index)

Mide eficiencia de costo. CPI = EV/AC. ≥1.0 está bajo presupuesto.

const { notify } = useNotifications();

notify({
  tipo: 'error',
  titulo: 'Falló',
  mensaje: 'No se pudo guardar',
  accion: { label: 'REINTENTAR', onClick: () => reintentar() }
});
