2 Design Tokens — el corazón del sistema

"Token" = constante con un nombre semántico. En lugar de escribir padding: '12px' en 200 lugares, escribimos SPACING.md. Si mañana queremos cambiar el spacing medio a 14px, lo cambiamos en UN lugar.

Categorías de tokens

SPACING

7 valores: xs · sm · md · lg · xl · 2xl · 3xl

Margenes, padding, gaps

TYPOGRAPHY

9 tamaños · 5 pesos · 5 tracking · 4 leading

Tamaños, pesos, espaciado

RADIUS

7 valores: sm → 3xl + full

Bordes redondeados

ANIMATION

3 duraciones · 4 easings

Transiciones consistentes

Z_INDEX

5 niveles: dropdown → tooltip

Apilamiento ordenado

BASE (extendido)

35+ colores incluyendo LPS

Paleta completa GRAPCO

Helpers ergonómicos

Junto a los tokens entrego 3 funciones helper que aplican estilos rápido:

// Aplicar tipografía rápido
<p style={font('lg', 'bold', BASE.navy)}>Título</p>

// Card consistente
<div style={cardStyle({ padding: SPACING.lg })}>...</div>

// Botón consistente
<button style={buttonStyle('gold', 'lg')}>Acción</button>

9 Recomendaciones de continuidad

Mejoras que recomiendo priorizar en próximos bloques.

🌙

Modo oscuro (dark mode)

Ya tenemos los tokens. Solo falta una capa theme con colores invertidos. Usuarios que trabajan de noche en obra lo agradecerán.

♿

Accesibilidad WCAG AA

Auditar contrastes de color, agregar aria-labels a botones con solo iconos, soportar prefers-reduced-motion.

📱

PWA installable

Agregar manifest + service worker para que capataces instalen GRAPCO como app en su celular, modo offline básico.

🎯

Storybook de componentes

Catalogar visualmente todos los componentes (button, card, chip, modal). Útil para que nuevos desarrolladores entiendan rápido el sistema.

🔔

Sistema de notificaciones

Reemplazar Toast actual por sistema más rico: notificaciones persistentes, agrupadas, con acciones inline (deshacer, ver más).

⌨️

Atajos de teclado

Cmd+K para command palette, Esc para cerrar modales, flechas para navegar tabs. Power users lo amarán.

📊

Tooltips contextuales

Hover sobre KPIs muestra explicación: "CPI = EV/AC. Mide eficiencia de costo, ≥1.0 está bajo presupuesto."

🔐

Seguridad Firebase (URGENTE)

Las reglas Firestore + Storage siguen abiertas. Cualquiera con la URL puede leer/escribir todo. Bloque 11 = solo seguridad.
