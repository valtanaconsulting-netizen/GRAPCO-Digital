🦺 MODULO CALIDAD

Protocolos por elemento, PETs, no conformidades, ensayos. Replica el formato Excel de GRAPCO con flujo digital.

Componentes del modulo

📋

Protocolos por elemento

Cada zapata, columna, viga, losa tiene su protocolo. Replica el formato Excel propio de GRAPCO. Estados: pendiente → liberado → observado → re-inspeccion. Firma residente + supervisor cliente directo en la app.

📜

PETs (Procedimientos Escritos de Trabajo Seguro)

Documentos paso a paso para cada actividad critica. Capataz/operario debe firmar que leyo el PET antes de empezar la actividad. Vinculado al LPS. Sin PET firmado → no se puede registrar avance.

🚨

No Conformidades (NC)

Cuando algo no cumple especificacion. Foto, descripcion, causa raiz, accion correctiva, fecha cierre. Vinculacion a partida y elemento BIM. Estado: abierta → tratamiento → cerrada → verificada por supervisor.

🧪

Ensayos de laboratorio

Resultados externos (rotura probetas, slump, granulometria, densidad). Upload PDF o ingreso manual. Vincular al elemento + fecha vaciado. Alertas automaticas si f'c < especificado.

📊

Dashboard Calidad

% liberacion vs avance · NCs abiertas · Protocolos pendientes · PETs vencidos · Resultados ensayos. Vista ejecutiva para gerencia de obra.

🔗

Integracion BIM

Click en un elemento del modelo Revit → ver protocolo + estado + ensayos + NCs asociadas. El visor BIM colorea: verde=liberado, ambar=pendiente, rojo=NC abierta.

📊 MODULO OFICINA TECNICA

RDO automatico, valorizaciones contractuales, adicionales/deductivos, cartas, cronograma vs real.

Componentes del modulo

📅

RDO Automatico

Reporte Diario de Obra que se autogenera al cierre del dia: clima (API), personal (de tareos), equipos, avance LPS, fotos del capataz, observaciones. Residente solo revisa, edita y firma. Genera PDF descargable.

💰

Valorizaciones contractuales

Mensual al cliente. Calcula automatico desde LPS: avance acumulado × precio unitario contractual. Aplica adelanto, deducciones, IGV. Genera expediente PDF formato OSCE/cliente.

📐

Reajuste polinomico (DS 011-79-VC)

Calcula automaticamente el factor de reajuste mes a mes con indices INEI cargados. Aplica a la valorizacion. Cero error manual.

➕

Adicionales y deductivos

Cambios al contrato. Cada adicional con sustento tecnico, presupuesto, aprobacion del cliente. Actualiza presupuesto base automaticamente.

📨

Cartas y comunicaciones

Bandeja entrada/salida con el cliente. Cada carta: fecha, asunto, PDF adjunto, vinculo a observacion/atraso. Trazabilidad para auditoria contractual.

📈

Curva S contractual vs real

Grafico maestro de obra. Curva programada vs ejecutada. Identifica atraso/adelanto. Calcula SPI a nivel obra (no solo HH).

Como se calcula la valorizacion

// Para cada partida del contrato:
avanceAcumActual = sumar metrados ejecutados desde inicio hasta fecha valorizacion
avancePeriodo = avanceAcumActual − avanceAcumValorizacionAnterior
montoBruto = avancePeriodo × precioUnitarioContractual

// Aplicar reajuste polinomico (si aplica)
factorReajuste = formula DS 011-79-VC con indices INEI del mes
montoReajustado = montoBruto × factorReajuste

// Aplicar adelantos y deducciones
amortizacionAdelanto = montoBruto × (% adelanto / 100)
multas = penalidades si aplica
subtotal = montoReajustado − amortizacionAdelanto − multas

// IGV
igv = subtotal × 0.18
total = subtotal + igv
